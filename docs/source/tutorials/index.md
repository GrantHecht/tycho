# Tutorials

Learning-oriented walkthroughs. Start here if you are new to Tycho — the
{doc}`first-VectorFunction tutorial </tutorials/basics/your_first_vectorfunction>`
builds you up from a single symbolic input to complete dynamics expressions, then
{doc}`propagating an ODE </tutorials/basics/propagating_an_ode>` integrates those
dynamics forward in time, and
{doc}`setting up a phase </tutorials/basics/your_first_phase>` hands them to the
optimizer. Once you have the basics, the
{doc}`worked examples gallery </tutorials/examples/index>` collects complete,
runnable programs you can adapt to your own problems.
