# Getting Started

Get Tycho built into a conda environment, confirm `import tychopy` works, then
write a real program: a self-differentiating dynamics model in a handful of
lines.

- {doc}`Installing Tycho <install>` — build the `_tychopy` extension into a
  conda environment and verify your setup.
- {doc}`Your first Tycho program <first_program>` — the five-minute path from a
  working import to a runnable VectorFunction with an automatic Jacobian.

From there, head to the
{doc}`first-VectorFunction tutorial </user_guide/first_vectorfunction>`
to go deeper.
