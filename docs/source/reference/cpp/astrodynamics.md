# Astrodynamics

The C++ API for Tycho's astrodynamics subsystem, declared in the
`tycho::astro` namespace and exposed through the `include/tycho/astro.h`
umbrella header. Python users should consult the
{doc}`Python reference </reference/python/astrodynamics>`.

The subsystem provides three element sets — Cartesian, classical (Keplerian),
and Modified Equinoctial Elements (MEE) — with numeric conversion/propagation
free functions templated on a scalar type, differentiable
{cpp:class}`tycho::vf::VectorFunction` types for embedding element conversions
inside optimal-control expression graphs, and dynamics VectorFunction classes
whose instances serve as the ODE of a phase.

## Coordinate conversions

Templated free functions that convert between element sets.  Each accepts a
`Vector6<Scalar>` (an `Eigen::Matrix<Scalar, 6, 1>`) and returns the converted
vector; `Scalar` may be `double` or an `Eigen::Array` SuperScalar type for
batched evaluation.

## Kepler propagator

The universal-variable Kepler propagator as a differentiable VectorFunction
(IR=7, OR=6): maps `[rx, ry, rz, vx, vy, vz, dt]` to the propagated
Cartesian state using the LCD (Stumpff-function) iteration with IFT-composed
analytic Jacobian and Hessian.

## VectorFunction conversions

Differentiable VectorFunction types for embedding element-set conversions
inside optimal-control expression graphs.  Unlike the free-function conversions
above, these carry analytic Jacobian and Hessian information.

```{eval-rst}
.. doxygenclass:: tycho::astro::CartesianToMEE
   :project: tycho
   :members:
```

`CartesianToClassic` and `ModifiedToCartesian` are generated by the
`BUILD_FROM_EXPRESSION` macro (which expands to a `struct` that inherits from
{cpp:class}`tycho::vf::VectorExpression`) and are not independently indexed by
Doxygen.  They are documented here by hand:

```{eval-rst}
.. cpp:struct:: tycho::astro::CartesianToClassic

   Cartesian → classical orbital elements VectorFunction (IR=6, OR=6).

   Accepts Cartesian state ``[rx, ry, rz, vx, vy, vz]`` and computes classical
   orbital elements ``[a, e, i, Omega, omega, M]``, where ``M`` is the elliptic
   mean anomaly for ``e < 1`` and the hyperbolic mean anomaly for ``e > 1``.
   Generated via ``BUILD_FROM_EXPRESSION`` from ``CartesianToClassic_Impl``
   (see ``include/tycho/detail/astro/conversions/kepler_utils.h``).

   Constructed with a gravitational parameter: ``CartesianToClassic vf(mu);``.

.. cpp:struct:: tycho::astro::ModifiedToCartesian

   MEE → Cartesian VectorFunction (IR=6, OR=6).

   Accepts Modified Equinoctial Elements ``[p, f, g, h, k, L]`` and returns
   Cartesian state ``[rx, ry, rz, vx, vy, vz]`` via direct closed-form
   conversion.  Generated via ``BUILD_FROM_EXPRESSION`` from
   ``ModifiedToCartesian_Impl``
   (see ``include/tycho/detail/astro/conversions/kepler_utils.h``).

   Constructed with a gravitational parameter: ``ModifiedToCartesian vf(mu);``.
```
