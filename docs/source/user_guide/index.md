# User Guide

The User Guide teaches Tycho by example. Concepts are woven together with
hands-on walkthroughs, task-oriented how-to recipes, and complete worked
examples — everything you need to learn the library and apply it to real
trajectory-design and optimal-control problems.

Start with **Learn the basics** to build up the core ideas — VectorFunctions,
phases and collocation, and the astrodynamics models — each paired with a
hands-on tutorial. Then reach for the **How-to guides** when you need a focused
recipe for a specific task.
